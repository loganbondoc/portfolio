A web application designed to generate workout programs and track performance by logging an individual’s repetitions, workout dates and rest times. This was created as a part of an assessment for the DECO2017 unit, which taught the basics of web development utilising HTML, CSS, Javascript and Node.js.

Smart Gate Case Study

Through investigating the Sydney Train System, our team found issues pertaining to comfortability, efficiency, safety and navigation. To combat this we designed the "Smart Gate", and conducted user testing upon its usability and effectiveness within the real world.
